Future Direction

My Career Goals

I want to work in IT or computer science. Both fields rely on computing and need strong programming skills. With my degree, I'll use high-level programming to understand complex systems, build algorithms, and create real solutions. I'm keen to apply these skills to solve problems and help organisations work better.

Skills I Want to Develop

There are a few areas I want to focus on as I grow:

  • Advanced programming (like Python and Java)
  • Data analysis and how to work with large datasets
  • Cybersecurity and how to protect systems
  • Cloud computing tools like AWS or Azure
  • Building websites and apps from scratch

Personal Projects

I'm planning to build a few projects to put my skills into practice. These will help me learn more and give me something real to show on my resume. Some ideas include:

  • A personal website or portfolio
  • A small app or tool that solves a simple problem
  • A dashboard that shows data in useful ways
  • Automating boring tasks using code

Long-Term Vision

In the future, I want to work in a role where I can keep learning and stay challenged. I see myself working in a team that builds useful tech, either in data, software, or systems. I also want to keep doing personal projects on the side, maybe even freelance or start something of my own one day.
